# landing-page
Projeto de landing page em TS(NextJS) e TailWind.
Este projeto tem o intuito servir como base de estudo e a empresa utilizada é fictícia.

Para baixar o projeto, clone a main branch na sua máquina com:
- git clone https://github.com/aryelbargas/landing-page.git

Com o projeto baixado, abra um terminal no diretório do projeto e rode:
- npm install && npm run dev
